A hands-on logistic regression walkthrough using the Pima Indians Diabetes dataset (768 female patients, 8 clinical features) to predict diabetes risk. The notebook covers data cleaning (replacing impossible zero-values with column means), EDA, and iterative feature selection by dropping insignificant predictors (p > 0.05) until four remain: Pregnancies, Glucose, BMI, and Pedigree. Multicollinearity is validated via VIF before interpreting coefficients as odds ratios — with the diabetes pedigree function showing the strongest effect (~184% increase in odds per unit). Model performance is evaluated using a confusion matrix, accuracy score, and ROC-AUC curve across a stratified 70/30 train/test split.
